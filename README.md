Stata help
==========

Firefox search plugin for the [Stata help](http://www.stata.com/help.cgi?contents). Open your explorer, navigate to

	%AppData%\Mozilla\Firefox\Profiles\

click on the subfolder of your profile, go to searchplugins\ and copy the file stata.xml there.
